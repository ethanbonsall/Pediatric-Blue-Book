# Pediatric Blue Book

##  Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **UI**: React, Tailwind CSS
- **Backend**: Supabase (used for project data and storage)
- **Deployment**: Vercel
- **Language**: TypeScript

## Running Locally

    git clone https://github.com/ethanbonsall/Pediatric-Blue-Book.git

    cd pediatric-blue-book
    
    cd web

    npm install --f

    npm run dev
