# TEAM O - COMP 523

##  Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **UI**: React, Tailwind CSS
- **Backend**: Supabase (used for project data and storage)
- **Deployment**: Vercel
- **Language**: TypeScript

## Running Locally

    git clone https://github.com/ethanbonsall/COMP-523-O.git
    
    cd web

    npm install

    npm run dev
